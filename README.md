Apa Itu Tipe Data?

Tipe Data merupakan pengklasifikasian data berdasarkan jenis data tersebut, atau mudahnya tipe data dapat juga disebut jenis data.

Terdapat 3 tipe data dalam python, yaitu :

Numerik

String

Boolean
